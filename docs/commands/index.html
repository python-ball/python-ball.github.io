echo

Normal Behavior

echo >*> >*> >*> ...
- The arguments are cast to strings and concatenated, then sent in the same channel the command was invoked from. 
echo >channel> >*> >*> >*> ...
- The arguments are cast to strings and concatenated, then sent in the specified channel.

Pipe Behavior

piped : string
- The piped value is echoed, regardless of any arguments given.
